Profil Singkat

Saya tertarik pada pertemuan antara pendidikan, software, dan elektronika. Saat tidak mengajar, saya biasanya bereksperimen dengan skrip Python kecil, web app sederhana, atau perangkat mikrokontroler untuk menguji ide dan menyiapkan materi belajar baru.
